# Assets

Questa cartella contiene le risorse grafiche per Connee Alarm.

## Logo
- **connee-logo.png** - Logo Connee nero (per sfondi chiari)
- **connee-logo-white.png** - Logo Connee bianco (per sfondi scuri)

### Utilizzo nella Dashboard
I loghi vengono caricati automaticamente dalla dashboard tramite URL GitHub raw:
- Light mode: https://raw.githubusercontent.com/conneehome/ajax/main/assets/connee-logo.png
- Dark mode: https://raw.githubusercontent.com/conneehome/ajax/main/assets/connee-logo-white.png
